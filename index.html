FAQ – Häufige Fragen
Wie nutze ich DexorEH?
Um DexorEH zu nutzen, kopieren Sie bitte den Loader-Code oben und fügen Sie ihn in einen unterstützten Roblox-Executor ein. Das Script lädt sich danach automatisch.
Für welche Spiele ist DexorEH gedacht?
DexorEH wurde speziell für das Roblox-Spiel „Emergency Hamburg“ entwickelt. Andere Games werden zurzeit nicht unterstützt.
Wird ein Key benötigt?
Für DexorEH ist aktuell kein Key oder Keysystem notwendig. Nach dem Einfügen des Scripts kann die Nutzung direkt erfolgen.
Welche Executor werden empfohlen?
Die Nutzung wurde mit Synapse X, Evon, Delta und Kiwo getestet. Andere bekannte Executor funktionieren in der Regel ebenfalls problemlos. Bitte beachten Sie die aktuellen Statusmeldungen zu Blacklists.
Ist DexorEH sicher zu benutzen?
DexorEH wurde nach bestem Wissen für die Community entwickelt. Ein Restrisiko gibt es immer beim Nutzen von Scripts in Roblox – nur von vertrauenswürdigen Quellen nutzen!
Empfohlen wird immer ein Alt-Account.
Was tun, wenn das Script nicht lädt?
Prüfe, ob du die aktuelle Loader-URL nutzt, dein Executor auf dem neuesten Stand ist und ob Roblox keine Verbindungsschwierigkeiten hat. Gegebenenfalls das Spiel und Script neu starten.
Kann ich DexorEH auf dem Handy nutzen?
Nein, derzeit wird ausschließlich PC unterstützt. Mobile-Executoren funktionieren (noch) nicht für DexorEH.
Gibt es regelmäßige Updates?
Ja, der Hub wird regelmäßig aktualisiert, besonders nach Emergency Hamburg Updates. Bugfixes oder neue Features kommen meistens kurzfristig dazu.
Kann ich für das Nutzen gebannt werden?
Die Nutzung von Scripts ist immer mit einem Ban-Risiko verbunden. DexorEH hat Schutzmechanismen, aber ist nie 100% detected-safe! Immer auf eigene Gefahr.
Wird mein Roblox-Account dadurch beeinflusst?
DexorEH verändert nichts dauerhaft an deinem Roblox-Account. Nach Rejoin oder ohne Executor ist alles wieder wie vorher.
Wer steckt hinter DexorEH?
Der Hub wurde von Roblox-Usern aus der Szene entwickelt, um Emergency Hamburg Usern die bestmöglichen Features zu bieten.
Support und Updates gibt’s nur über den offiziellen Discord.
Ich habe einen Bug gefunden – wohin melden?
Melde Bugs und Probleme einfach direkt über den Discord oder die im Hub angegebene Support-Kontaktmöglichkeit.
Wie kann ich updaten?
Immer einfach den Loader nutzen! Die aktuelle Version wird dann geladen. Ein manuelles Update ist nicht nötig.
Gibt es Einschränkungen bei Roblox-Events?
Manche temporären Event-Änderungen können bestimmte Funktionen beeinträchtigen. In solchen Fällen gibt es meistens zeitnah einen Fix per Hub-Update.
Warum laden einige Funktionen manchmal nicht?
Das kann an Roblox-Serverlag, Executor-Einstellungen oder Updates im Spiel liegen. Einfach neu ausführen oder Support fragen.
Wie kann ich Feedback geben?
Feedback und Feature-Wünsche werden am besten über Discord oder das offizielle GitHub angenommen.
Was, wenn ein Update den Hub blockiert?
Meistens gibt’s in wenigen Stunden einen Fix. Auf Discord werden solche Infos am schnellsten geteilt.
Hinweis: DexorEH ist ein reines Fanprojekt. Für Probleme, Ideen oder Support – Discord oder GitHub checken!
